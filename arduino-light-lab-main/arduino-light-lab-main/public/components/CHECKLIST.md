## üé® Quick Creation Tips

### For LEDs (Green, Blue, RGB)
- Copy `led-red.svg` and change colors:
  - Green: `#10B981` (fill) `#065F46` (stroke)
  - Blue: `#3B82F6` (fill) `#1E3A8A` (stroke)
  - RGB: Multiple colored sections

### For Resistors (1K, 10K)
- Copy `resistor-220.svg` and change color bands:
  - 1KŒ©: Brown-Black-Red-Gold
  - 10KŒ©: Brown-Black-Orange-Gold

### For Sensors
- Use rectangular/circular shapes
- Add 3-4 pin connections
- Use realistic colors from actual component photos

### For Displays
- Rectangle with dark background
- Grid pattern for pixels/characters
- Pin row at bottom
